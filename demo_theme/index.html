Library of plugins

They will help you to faster your development, to add functionality and to decorate the interface. It is ok for beginners and professionals. I hope this collection will help you. If you have any questions you can connect with me to social networks.

Animation

snabbt.js

Fast animations with Javascript and CSS transforms(Work in progress)

http://daniel-lundin.github.io/snabbt.js/

https://github.com/daniel-lundin/snabbt.js

AniJS

A Library to Raise your Web Design without Coding. The most easy and quick way to Raise your web!!!

http://anijs.github.io/

https://github.com/anijs/anijs

Effeckt.css

A Performant Transitions and Animations Library

https://github.com/h5bp/Effeckt.css

Hover.css

A collection of CSS3 powered hover effects to be applied to links, buttons, logos, SVG, featured images and so on. Easily apply to your own elements, modify or just use for inspiration. Available in CSS, Sass, and LESS.

http://ianlunn.github.io/Hover/

https://github.com/IanLunn/Hover

jQueryTween

This plugin is a bridge to tween.js and aims to drastically simplify your work. Each script does what it does best, for high performance hardware accelerated animations.

http://thednp.github.io/jQueryTween/

cta.js

cta.js or "Call to Animation" is a light-weight performant library to animate any element ("action") onto any other element ("effect") on the page.

http://kushagragour.in/lab/ctajs/

https://github.com/chinchang/cta.js

Graphics

Lazy Line Painter

A jQuery plugin for path animation using the CSS.

http://lazylinepainter.info/

https://github.com/camoconnell/lazy-line-painter

SVG Morpheus

JavaScript library enabling SVG icons to morph from one to the other. It implements Material Design's Delightful Details transitions.

http://alexk111.github.io/SVG-Morpheus/

https://github.com/alexk111/SVG-Morpheus

Arbor.js

Arbor is a graph visualization library built with web workers and jQuery. Rather than trying to be an all-encompassing framework, arbor provides an efficient, force-directed layout algorithm plus abstractions for graph organization and screen refresh handling.

http://arborjs.org/

https://github.com/samizdatco/arbor

Chartist.js

Chartist's goal is to provide a simple, lightweight and unintrusive library to responsively craft charts on your website. It's important to understand that one of the main intentions of Chartist.js is to rely on standards rather than providing it's own solution to a problem which is already solved by those standards.

http://gionkunz.github.io/chartist-js/

https://github.com/gionkunz/chartist-js

JointJS

JointJS is a JavaScript diagramming library. It can be used to create either static diagrams or, and more importantly, fully interactive diagramming tools and application builders.

http://jointjs.com/

https://github.com/clientIO/joint

Epoch

Epoch is a general purpose charting library for application developers and visualization designers. It focuses on two different aspects of visualization programming: basic charts for creating historical reports, and real-time charts for displaying frequently updating timeseries data.

http://fastly.github.io/epoch/

https://github.com/fastly/epoch

dc.js

Dimensional charting built to work natively with crossfilter rendered using d3.js. Check out the example page with a quick five minutes how to guide. For a detailed API reference and more please visit the Wiki.

https://github.com/dc-js/dc.js

vis.js

Vis.js is a dynamic, browser based visualization library. The library is designed to be easy to use, handle large amounts of dynamic data, and enable manipulation of the data.

https://github.com/almende/vis

sigma.js

Sigma is a JavaScript library dedicated to graph drawing.

http://sigmajs.org/

https://github.com/jacomyal/sigma.js

Highcharts JS

Highcharts JS is a JavaScript charting library based on SVG and VML rendering.

http://www.highcharts.com/

https://github.com/highslide-software/highcharts.com

jsPlumb

jsPlumb provides a means for a developer to visually connect elements on their web pages. It uses SVG or Canvas in modern browsers, and VML on IE 8 and below.

https://jsplumbtoolkit.com/demo/flowchart/dom.html

https://github.com/sporritt/jsplumb/

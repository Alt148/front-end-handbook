Graphics

Lazy Line Painter

A jQuery plugin for path animation using the CSS.

http://lazylinepainter.info/

https://github.com/camoconnell/lazy-line-painter

SVG Morpheus

JavaScript library enabling SVG icons to morph from one to the other. It implements Material Design's Delightful Details transitions.

http://alexk111.github.io/SVG-Morpheus/

https://github.com/alexk111/SVG-Morpheus

Arbor.js

Arbor is a graph visualization library built with web workers and jQuery. Rather than trying to be an all-encompassing framework, arbor provides an efficient, force-directed layout algorithm plus abstractions for graph organization and screen refresh handling.

http://arborjs.org/

https://github.com/samizdatco/arbor

Chartist.js

Chartist's goal is to provide a simple, lightweight and unintrusive library to responsively craft charts on your website. It's important to understand that one of the main intentions of Chartist.js is to rely on standards rather than providing it's own solution to a problem which is already solved by those standards.

http://gionkunz.github.io/chartist-js/

https://github.com/gionkunz/chartist-js

JointJS

JointJS is a JavaScript diagramming library. It can be used to create either static diagrams or, and more importantly, fully interactive diagramming tools and application builders.

http://jointjs.com/

https://github.com/clientIO/joint

Epoch

Epoch is a general purpose charting library for application developers and visualization designers. It focuses on two different aspects of visualization programming: basic charts for creating historical reports, and real-time charts for displaying frequently updating timeseries data.

http://fastly.github.io/epoch/

https://github.com/fastly/epoch
